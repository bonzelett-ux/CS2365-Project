1 create acc

4 view cart
5 make order
6 view my orders
7 browse catalog 
8 log out
9 exit
type one of these nums and hit enter to do that process.

1- create an account (already explained)
2- log in (already explained)
3- allows you to select items from the catalog to put in the cart (enter the item num, then the quantity u want to use, type done when youre done selecting items)

7- allows you to see the catalog even if you aren't logged in (no additional input required)
