Project Part 2 (GUI):

For this part of the project the UML is quite similar, the only changes i made were in customer order system and order system where methods and fields were edited. Other than that everything else is the same. 

Running the GUI:

ensure the files are all part of the same package (rename the package at the top of the files to whatever package you are using too)

Once you hit run, you will be presented with a main menu where you will be able to see 9 buttons to choose what to do (very similar to my project 2 format but numbers switched around so it looks better with the greyed out buttons). Some buttons will be greyed out since you wont be logged in yet. First you will need to create an account by pressing button 1. This will bring up a menu where you can input your customer ID, password, name (you can type first and last in one name here i didn't make 2 separate fields for this), address, credit card, select a sec question, and type a sec q answer.
Here are some quick ones to input:
a
Abcd1234!
bob brown
a
CHOOSE ONE: 1234123412341234 ; 1111111111111111 ; 1234567890123456
(select any sec q)
a
Then select back to go back to main menu.

Select button 2 to log in. Log in by typing the customer id, password, and sec q answer you just made. Hit back to get back to main menu.
Now you can select any of the 9 buttons as none will be greyed out. 

1- create an account (already explained)
2- log in (already explained)
3- allows you to see the catalog even if you aren't logged in 
press back to go back to main menu

4- allows you to select items from the catalog to put in the cart 
Type the product id and quantity, then hit add to cart and continue until you are done, then hit back to go to main menu

5- allows you to see the current cart 
press back to go back to main menu

6- allows you to make the order if you have enough money on your card
select mail or pick up from the dropdown menu then hit confirm order
once done press back to go back to main menu

7- allows you to see the orders you already made 
hit back to go back to main menu

8- logs you out 

9- ends the program
